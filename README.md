# Sky: Children of the Light — Localization Tool 📝 Light Mod Kit

Удобный, быстрый и безопасный текстовый редактор на **PyQt6**, созданный специально для моддинга и редактирования файлов локализации (`Localizable.strings`) в ПК-версии игры *Sky: Children of the Light*.

Интерфейс утилиты спроектирован в стиле Visual Studio Code (Dark Theme), обеспечивая максимальный комфорт при работе с огромными текстовыми массивами игры.

---

## 🌟 Главные фичи инструмента:

- **Стиль VS Code:** Полноценный темный режим (Dark Mode) и моноширинный шрифт Consolas, от которого не устают глаза.
- **Умная нумерация строк:** Слева расположена интерактивная колонка с номерами строк (1, 2, 3...), которая идеально синхронизирована с прокруткой текста.
- **Продвинутый поиск (Ctrl + F):** Вызываемая панель поиска на лету подсвечивает все совпадения приятным желтым цветом по всему файлу и позволяет прыгать по результатам кнопкой "Далее" или клавишей Enter.
- **Подсветка игрового синтаксиса:** Программа автоматически окрашивает ключи, значения строк, кавычки, переносы строк (`\n`), переменные (`{{1}}`) и служебные XML-теги игры (`<2>`, `<b>`, `</warning>`). Вы сразу видите структуру кода и не сотрете лишнего!
- **Железобетонная безопасность (.bak):** Перед каждым сохранением программа автоматически делает резервную копию оригинального файла игры. Если вы случайно сломаете синтаксис — игра не повредится, вы всегда сможете откатиться.
- **Отказоустойчивость:** Все процессы чтения и записи обернуты в блоки обработки ошибок. Программа не вылетает молча, а всегда выведет понятное окно с ошибкой, если файл занят системой или защищен.

## 🛠 Установка зависимостей

Для работы программы требуется библиотека PyQt6. Установите её через командную строку Windows:

```bash
pip install PyQt6 --break-system-packages
```

---

## 🚀 Как использовать:

1. Откройте файл `sky_localization_tool.py` через любой текстовый редактор.
2. В строке №14 в переменной `DEFAULT_STRINGS_PATH` укажите ваш стандартный путь к папке игры Sky, чтобы программа сразу открывалась в нужном месте.
3. Запустите скрипт:
   ```bash
   python sky_localization_tool.py
   ```
4. Нажмите **"Открыть Localizable.strings"**, вносите любые правки, используйте **Ctrl + F** для быстрого поиска нужных фраз, и жмите **"Сохранить и применить в игре"**.

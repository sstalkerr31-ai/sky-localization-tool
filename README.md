# Sky: Children of the Light — Localization Tool 📝 Light Mod Kit

Удобный, быстрый и полностью автономный графический редактор на **PyQt6**, созданный специально для моддинга, перевода и редактирования файлов локализации (`Localizable.strings`) в ПК-версии игры *Sky: Children of the Light*.

Интерфейс утилиты спроектирован в стиле Visual Studio Code (Dark Theme). Больше **НЕ НУЖНО** открывать скрипт через Блокнот и вручную прописывать пути в коде — всё управление полностью кнопочное!

---

## 🌟 Главные фичи инструмента:

- **Стиль VS Code:** Полноценный темный режим (Dark Mode) и моноширинный шрифт Consolas, от которого не устают глаза.
- **Умная нумерация строк:** Слева расположена интерактивная колонка с номерами строк (1, 2, 3...), которая идеально синхронизирована с прокруткой текста.
- **Продвинутый поиск (Ctrl + F):** Нажмите **Ctrl + F**, чтобы мгновенно вызвать выезжающую панель поиска! Она на лету подсвечивает все совпадения приятным желтым цветом по всему файлу и позволяет прыгать по результатам кнопкой "Далее" или клавишей Enter.
- **Подсветка игрового синтаксиса:** Программа автоматически окрашивает ключи, значения строк, кавычки, переносы строк (`\n`), переменные (`{{1}}`) и служебные XML-теги игры (`<2>`, `<b>`, `</warning>`). Вы сразу видите структуру кода и не сотрете лишнего!
- **Полностью кнопочный интерфейс:** В верхней панели доступны 4 удобные кнопки — **Открыть**, **Сохранить**, **Найти** и **Папка игры**.
- **Динамическая настройка путей:** Кнопка "⚙️ Папка игры" позволяет выбрать директорию со Sky прямо через проводник Windows. Программа запомнит ваш выбор в файл `settings.txt` и при следующих запусках сразу откроет нужную папку.
- **Железобетонная безопасность (.bak):** Перед каждым сохранением программа автоматически делает резервную копию оригинального файла игры. Если вы случайно сломаете синтаксис — игра не повредится, вы всегда сможете откатиться.
- **Отказоустойчивость:** Все процессы чтения и записи обернуты в блоки обработки ошибок. Программа не вылетает молча, а всегда выведет понятное окно с ошибкой, если файл занят системой или защищен.

## 🛠 Установка зависимостей

Для работы программы требуется библиотека PyQt6. Установите её через командную строку Windows:

```bash
pip install PyQt6 --break-system-packages
```

---

## 🚀 Как использовать:

1. Запустите скрипт:
   ```bash
   python sky_localization_tool.py
   ```
2. При первом запуска нажмите кнопку **"⚙️ Папка игры"** и выберите папку, где лежат файлы локализации вашей игры Sky.
3. Нажмите **"📂 Открыть"** (программа сразу откроет вашу сохраненную папку) и выберите файл `Localizable.strings`.
4. Вносите любые правки. **Для быстрого поиска нужных фраз обязательно используйте горячие клавиши Ctrl + F!**
5. Нажмите **"💾 Сохранить"** для мгновенного применения изменений прямо в папке игры.
